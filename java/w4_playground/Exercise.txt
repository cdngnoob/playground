SC P4.3 Collections Spielwiese
======================
Wir haben euch hier eine sehr einfache und nicht ganz vollständige Implementierung eines Stacks und einer Queue vorbereitet.
Schaut euch den Code an und spielt damit herum. Die Fortgeschritteneren unter euch können versuchen diese Collections so umzubauen, dass die enthaltenen Elemente von einem beliebigen Typ sein können. Seht euch dazu die <a href="https://docs.oracle.com/javase/tutorial/java/generics/index.html" target="_blank">Java Generics</a> an. Ein anderes interessantes Thema zu diesem Bereich sind die <a href="http://openbook.rheinwerk-verlag.de/javainsel9/javainsel_13_002.htm#mj6a20dd38f9c93abb1c32b6f2f56357af">Iteratoren</a>.
